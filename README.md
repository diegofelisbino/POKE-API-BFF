# POKE-API-BFF [Em desenvolvimento]
Uma API desenvolvida em .NET para servir como um BFF para dois clientes distintos, com necessidades especificas.

Tecnologias utilizadas no projeto:

- AutoMapper
- Refit
- Log
  - Serilog
  - Health Checks
  - ElmahIO
  - Sentry
- Versionamento
  - Swagger

Tests:
  - xUnit
    - Testes Unitários 
    - Testes de Integração
    - Testes de Mutação
      - Stryker.Net
